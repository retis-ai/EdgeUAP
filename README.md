# Edge-Only Universal Adversarial Attacks (Demo)

This repository provides a **demo implementation** of the attacks proposed in the paper:

> **Edge-Only Universal Adversarial Attacks in Distributed Learning**  
> Giulio Rossolini, Tommaso Baldi, Alessandro Biondi, Giorgio Buttazzo  
> *(under review)*
> 
This demo shows how to run **edge-only Universal Adversarial Perturbation (UAP)** attacks on classical CNNs in a split inference setting.

## Installation
Create your venv from requirements.txt
